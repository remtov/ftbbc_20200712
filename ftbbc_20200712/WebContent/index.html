성서침례교회란?

성서침례교회는 미국의 최대 교단인 남침례교회(South Baptist Convention)에 그 역사를 두고 있다. 19세기 말부터 시작한 자유주의 사상에 비판적인 시각을 가지고 있던 프랭크 노리스 (Frank Norris : 1877~1952)가 남침례교회에서 분리되어 나오게 되었다. 그 후 G.B.Vick박사를 중심으로 뭉친 120여명의 목사들이 1950년 5월 24일 미국 텍사스주의 포트 워스(Fort Worth)에서 성경중심적이고 철저히 보수적인 교회들을 세우겠다는 각오로 시작된 것이 성서침례교회의 시작이다.

신앙선언문

우리의 믿음

01. 우리는 신구약 정경(正經) 66권은 성령의 영감으로 기록된 하나님의 말씀이며, 성경은 우리의 신앙과 실천에 있어서 최고(最高)의 권위(權威)이며, 성경이 가르치는 모든 내용은 절대 무오 (無誤)함을 믿는다.
마태복음 5장 17~18절 / 요한복음 10장 35절 / 디모데후서 3장 16절 / 베드로후서 1장 21절 더보기

02. 우리는 성경에는 여러 시대가 나타나 있으며,성경은 문자적(文字的), 문법적(文法的), 역사적 (歷史的) 원리에 의해 해석(解釋)해야 함을 믿는다.
로마서 16장 25~26절 / 에베소서 3장 1~6절 /골로새서 1장 24~29절

03. 우리는 스스로 존재하셔서 천지와 우주 만물 및 인간을 창조하신 창조주(創造主) 하나님을 믿는다.
창세기 1장 1, 26~27절 / 창세기 2장 2~3절 / 시편 8편 3절 / 출애굽기 3장 14절

04. 우리는 성삼위일체(聖三位一體) 하나님을 믿는다.
이사야 48장 16절 / 마태복음 28장 19~20절

05. 우리는 하나님의 독생자(獨生子)이신 예수 그리스도께서 완전한 하나님이시며 동시에 완전한 인간이시고, 그분만이 우리의 유일한 구세주(救世主)이심을 믿는다. 우리는 또한 그분의 동정녀(童貞女)탄생, 무죄(無罪)한 생애, 대속적(代贖的) 죽음,육체적 부활(復活), 승천 및 가시적 재림(再臨)을 믿는다.
요한복음 1장 1절 / 누가복음 2장 52절 / 이사야 7장 14절 / 마태복음 1장 18~25절 / 요한복음 20장 11~17절 / 고린도전서 15장 1~19절 / 사도행전 1장 11절 / 히브리서 4장 15절 / 요한일서 2장 2절

06. 우리는 신비체험과 관련된 오늘날의 은사운동 (방언, 신유, 入神, 透視, 靈書 등)은 비성경적인 것으로 믿는다.
고린도전서 13장 8~13절 / 데살로니가후서 2장 9~10절

07. 우리는 천사(天使)가 타락(墮落)하여 사탄(마귀,魔鬼)이 되었으며, 그와 함께 타락한 다른 천사들이 귀신(鬼神)이 되었음을 믿는다.
에스겔 28장 11~15절 / 이사야 14장 12~20절 / 마태복음 12장 24절

08. 우리는 하나님의 형상(形狀)대로 하나님에 의해 친히 창조된 인간은 죄 때문에 타락해 스스로 구원(救援)할 수 있는 능력이 전혀 없음을 믿는다.
로마서 5장 8절 / 로마서 6장 23절 / 에베소서 2장 1절 / 디모데전서 1장 15절

09. 우리는 예수 그리스도를 믿음으로 의롭다 함을 얻고 구원(救援)받음을 믿는다.
사도행전 15장 11절 / 갈라디아서 2장 16절 / 에베소서 2장 8~9절

10. 우리는 예수 그리스도께서 전 인류를 구속(救贖)하기 위해 십자가에 달려 죽으셨음을 믿는다.
요한복음 1장 29절 / 요한복음 3장 16절 / 고린도후서 5장 19절 / 디모데전서 4장 10절 / 베드로후서 2장 1절

11. 우리는 한번 구원받은 성도는 영원히 멸망(滅亡)하지 않음을 믿는다.
요한복음 10장 28~29절 / 로마서 8장 31~39절

12. 우리는 구원(救援)받은 신자(信者)가 침례(浸禮)에 순종함으로 개 교회의 회원(會員)이 됨을 믿는다.
사도행전 2장 41절 / 마태복음 28장 19~20절

13. 우리는 교회의 직분(職分)은 목사(牧師)와 집사(執事)뿐이며, 이는 남자에 국한됨을 믿는다.
디모데전서 3장 1~13절 / 디모데전서 2장 11~12절 / 고린도전서 14장 34~36절

14. 우리는 믿는 자의 침례와 주의 만찬만이 교회의 의식(儀式)임을 믿는다.
사도행전 2장 41절 / 고린도전서 11장 17~34절

15. 우리는 개교회의 독립성(獨立性)을 믿는다.
사도행전 13장 2~3절 / 로마서 12장 16절 / 고린도전서 1장 10절 / 고린도전서 6장 1~8절 / 고린도후서 13장 11절 / 에베소서 4장 3절

16. 우리는 정교(政敎)의 분리(分離)를 믿는다.
마태복음 22장 21절 / 로마서 13장 1~7절

17. 우리는 교회와 이스라엘은 구별됨을 믿는다.
고린도전서 10장 32절

18. 우리는 주님의 지상명령(至上命令)인 세계선교(世界宣敎)를 위해 신앙을 같이하는 교회들이 함께 협력해야 함을 믿는다.
마태복음 28장 18~20절

19. 우리는 교리의 일치에 근거하지 않는 에큐메니칼 운동(교회일치운동)은 비성경적인 것임을 믿는다.
고린도후서 6장 14~18절 / 요한계시록 18장 4절

20. 우리는 휴거(携去)가 7년 환란(患亂)전에 이루어지고 그리스도의 지상 재림 후에 천년왕국 (千年王國)이 세워짐을 믿는다.
데살로니가전서 1장 10절 / 데살로니가전서 4장 13~18절 / 요한계시록 3장 10절 / 요한계시록 20장 1~10절

21. 우리는 천국(天國)과 지옥(地獄)의 실재(實在)를 믿는다.
사도행전 7장 56절 / 에베소서 2장 6절 / 마가복음 9장 42~50절 / 요한계시록 20장 11~15절 / 요한계시록 21장 8절
